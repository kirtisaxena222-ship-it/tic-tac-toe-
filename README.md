# ❌⭕ Tic Tac Toe (C++)

A simple 2-player console-based Tic Tac Toe game built using C++.

## Features
- 2 Player Mode (X vs O)
- Win Detection
- Draw Detection
- Input Validation

## Concepts Used
- Arrays (2D Array)
- Functions
- Loops
- Conditional Statements
- Basic Game Logic

## How to Run

Compile:
g++ main.cpp -o tictactoe
screen shot
X | O | X
-----------
 O | X | O
-----------
 X |   | O
Run:
./tictactoe

# 🎮 Tic Tac Toe (C++)

## 📌 Description
This is a simple Tic Tac Toe game built using C++.  
Two players can play the game in the console.

---

## 🚀 Features
- Two-player mode  
- Turn-based gameplay  
- Winner detection  
- Draw detection  

---

## 🛠️ Tech Used
- C++

---

## ▶️ How to Run
1. Compile the program:
   g++ filename.cpp -o game

2. Run the game:
   ./game
